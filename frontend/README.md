# Realtime Chat Application

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- Socket.io

## Features
- Real-time messaging
- Message history
- MongoDB storage
- Timestamps
- REST APIs

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

```
PORT=5000
MONGO_URI=your_mongodb_uri
```

## APIs

GET /api/messages

POST /api/messages
